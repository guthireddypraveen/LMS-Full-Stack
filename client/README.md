# LMS Frontend

React + Vite frontend for the Learning Management System.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file with:
- VITE_BACKEND_URL
- VITE_CLERK_PUBLISHABLE_KEY  
- VITE_STRIPE_PUBLISHABLE_KEY
- VITE_CURRENCY

See `.env.example` for reference.
